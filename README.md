# 肌悩み別 成分診断

「効きそう」より、**効く理由（成分）**で選ぶ——肌悩みをひとつ選ぶと、皮膚科学の根拠にもとづいた
おすすめ成分・市販アイテム・研究データ（出典）を提案する、静的サイトです。

- 完全な静的サイト（HTML / CSS / JS）。ビルド不要、そのまま GitHub Pages で公開できます。
- 内容は一次情報（PubMed 等）でファクトチェック済み。出典は `ナレッジ/99_ファクトチェックと修正方針.md` を参照。

## ファイル構成

```
index.html              … トップ（ヒーロー・診断・UV土台・科学解説・フッター）
assets/
  css/style.css         … デザイン（5色トークン / レスポンシブ / アニメ）
  js/data.js            … ★編集ポイント：成分・効能・注意点・研究データ・商品・アフィリリンク
  js/app.js             … 描画・診断・※研究データ開閉・アニメーション
  img/
    ogp.png             … OGP画像 1200×630
    ogp.svg / favicon.svg
    products/           … （任意）商品の実写真を置く場所
ナレッジ/               … 元記事＋ファクトチェック結果（サイトには未使用の資料）
```

## ローカルで確認する

`index.html` をブラウザで開くだけでも動きます。うまく動かない場合は簡易サーバーで：

```bash
# Python があれば
python -m http.server 8000
# → http://localhost:8000 を開く
```

## アフィリエイトリンクの差し替え（最重要）

`assets/js/data.js` の先頭を編集するだけで、全リンクが一括で切り替わります。

1. **アソシエイトID** … `const AFFILIATE_TAG = "PLACEHOLDER-22";` を、あなたのIDに変更（例 `"yourid-22"`）。
2. **商品ページ直リンクにしたい場合** … 各商品の `asin: null` に正しい ASIN を入れる。
   - 例：`asin: "B07XXXXওXX"` → `https://www.amazon.co.jp/dp/ASIN/?tag=あなたのID`
   - `asin` が `null` のままなら、商品名での Amazon 検索リンク（リンク切れしない）になります。

## 商品画像を実写真にする（任意）

各商品はいま、カテゴリ別のライン画像（軽量SVG）を表示しています。実写真にするには：

1. 画像を `assets/img/products/` に置く（**大きい画像は webp に変換してから**。目安：横幅 400px 程度・100KB 以下）。
2. `data.js` の該当商品に `img: "products/ipsa.webp"` のように指定。

> Amazon の商品画像を直接転載するには PA-API 等の規約順守が必要です。自分で撮影した写真か、規約に沿った素材を使ってください。

## OGP（SNSシェア画像）のURL更新

公開ドメインが決まったら、`index.html` の `og:url` / `og:image` / `twitter:image` / `canonical` の
`https://tmk4men.github.io/skincare-shindan/` を実際のURLに置き換えてください。
OGP画像を作り直す場合は `assets/img/ogp.svg` を編集 → PNG化（1200×630）。

## GitHub Pages で公開

このリポジトリはすでに Pages 設定済みです（`main` ブランチ / ルート）。
内容を更新したら `git push` するだけで反映されます（反映に数十秒）。

新規に設定する場合：Settings → Pages → Source を「Deploy from a branch」、Branch を `main` / `/ (root)` に。

## コンテンツの方針（守ること）

- 化粧品で医薬品的な効果を**断定しない**（「シミが消える」「シワを治す」は不可）。「美白」「シワ改善」は医薬部外品の承認範囲の表現。
- アフィリエイト＝広告なので **PR表記**を明示（フッターに記載済み）。
- 効果には個人差がある旨・パッチテスト・受診目安などの**免責**を記載（フッターに記載済み）。

詳細な科学的根拠と、元記事からの修正点は `ナレッジ/99_ファクトチェックと修正方針.md` を参照。
